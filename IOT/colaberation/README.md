## FE

### 터치

입력이 언제 들어올지 모르는 특성상 iot의 파이썬 파일이 웹에서 받아온 JS 파일 혹은 함수를 가져오는 방식으로 구현할 예정

iot가 받은 화면 데이터(JS)들이 어디에 저장되는지 알아내야 한다.

### 얼굴인식

js에서 iot의 파이썬 파일을 호출하는 방식으로 적용할 예정

파이썬 파일은 카메라에서 얻은 이미지 데이터를 활용하여 와 BE의 얼굴인식 모델을 사용할 예정

위의 자세한 내용은 아래 BE 협의 내용을 참고

## BE

### 얼굴인식

iot의 웹캠에서 찍은 이미지 데이터를 BE로 보내고 BE에서 얼굴인식 모델을 통해 누구인지 인식한 후 다시 iot로 응답해주는 방식으로 진행할 예정

iot에서 직접 api를 사용하는 방법이 있는데
